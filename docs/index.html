🚀 Quick Install

# Clone repository
git clone https://github.com/iamfaz0/secprobe.git
cd secprobe

# Install dependencies
pip3 install -r requirements.txt

# Run SecProbe
python3 secprobe.py --help

Or use the one-line installer:

bash install.sh
